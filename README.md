# AI/ML Engineer Portfolio

A single-page personal portfolio website showcasing my work as an AI/ML Engineer — built with HTML, CSS, and vanilla JavaScript.

## About

This portfolio highlights my experience in machine learning, computer vision, and NLP, including project work, technical skills, and ways to get in touch.

## Features

- Responsive hero section with intro, tagline, and call-to-action buttons
- About section with bio and specialization
- Skills grid showcasing tools and technologies
- Projects section with descriptions, tech stack tags, and links
- Contact section with a message form and social links
- Smooth scroll and fade-in animations
- Fully responsive across desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3 (Flexbox/Grid, custom properties)
- Vanilla JavaScript

## Project Structure

```
├── index.html      # Main page structure
├── styles.css      # Styling and layout
├── script.js       # Interactivity and animations
├── assets/         # Images and icons
└── README.md       # Project documentation
```

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser — no build step or dependencies required

To make changes:
- Edit content directly in `index.html`
- Update styles in `styles.css`
- Update interactivity in `script.js`

## Deployment

This site is static and can be deployed for free on:
- GitHub Pages
- Vercel
- Netlify
